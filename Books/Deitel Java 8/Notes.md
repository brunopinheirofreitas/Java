**Control Instruction**

- Three types: 
  - Sequency
  - Selection
    - if, if-else, switch, else if
  - Repetition
    - for, while, do-while

*if*
```
if (condition){
   statement;
}
```` 

*if-else*
```
if (condition){
   statement;
} else {
   statement;
}
``` 
*else if*
```
if (condition){
   statement;
} else if (condition) {
   statement;
} else if (condition) {
   statement;
} else {
   statement;
}
```

*?:*
```
(booleanExpression ? statement if true : statement if false)
Example:
println(StudentGrade>=60 ? "Passed" : "Failed");
```

*while*
```
while (condition) {
   statement;
}
```
